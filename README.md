# French Verb Conjugations Reviewer
Tests you on the conjugations of verbs in French.

## Requirements
`verbs.txt` which contains rows of whitespace separated verbs in the following order:
- Infinitive
- je
- tu
- il/elle
- nous
- vous
- ils/elles
- Imperative (tu/il)
- Imperative (nous)
- Imperative (vous)
